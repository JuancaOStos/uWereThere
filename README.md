# uWereThere
![image](https://github.com/user-attachments/assets/8aed8228-62a2-4289-b291-bb0959441301)

### uWereThere is a social media where you can publish you favorite locations with others and discover new places you didn't know around you.

### You will be able to create a user account and start posting your favorite locations.

![image](https://github.com/user-attachments/assets/ccf4752f-76cb-41ef-8c81-cf0ac261ed9c)

![image](https://github.com/user-attachments/assets/79440f0d-745a-4e75-82cb-5df9e23b9a73)

![image](https://github.com/user-attachments/assets/a2b45785-eaae-4666-8085-320ffd18436f)

![image](https://github.com/user-attachments/assets/6d623222-6e5a-4f46-acb0-5927e8a90634)

### Once you log in, you will be able to see all the locations that have already been posted and all the users on the platform. You will be able to follow them, rate their posts, write comments on them and, of course, post your own locations.
